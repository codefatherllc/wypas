# Wypas

Game client releases for Wypas Reborn.

Download the latest release for your platform from the [Releases](https://github.com/codefatherllc/wypas/releases) page.

| Platform | File | Install |
|----------|------|---------|
| Windows  | `wypas-windows.exe` | Run directly |
| macOS    | `wypas-macos.dmg` | Drag to Applications |
| Linux    | `wypas-linux` | `chmod +x wypas-linux && ./wypas-linux` |

Native clients auto-update via the in-game updater.
